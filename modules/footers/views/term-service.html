เงื่อนไขการใช้งาน Pretshy

เงื่อนไขการใช้บริการของ Platform Pretshy

ข้อตกลงต่อไปนี้ (“ข้อตกลงการใช้งาน”) จะเป็นข้อตกลงการใช้งานของคุณกับ Pretshy , Pretshy IOS Application, Pretshy Android Application ซึ่งรวมถึงเนื้อหา การใช้เงิน และการให้บริการทั้งหมดบน Pretshy (“Platform”) ที่คุณต้องปฏิบัติตาม

การใช้ Platform สร้างบัญชีผู้ใช้งาน หรือกดยอมรับข้อตกลงการใช้งานของเรา จะถือว่าคุณยอมรับที่จะปฏิบัติตาม และอยู่ภายใต้ข้อตกลง รวมถึงยอมรับนโยบายการเป็นส่วนตัวที่เรากำหนด คุณจะไม่สามารถใช้บริการP latform ของเราได้ หากคุณไม่เห็นด้วยกับข้อตกลง หรือนโยบายความเป็นส่วนตัวดังกล่าว

Platform นี้เปิดให้บริการสำหรับผู้ใช้ที่มีอายุเกิน 18 ปีบริบูรณ์แล้วเท่านั้น คุณจะไม่สามารถใช้บริการ Platformข องเราได้ หากคุณมีอายุต่ำกว่า 18 ปี การใช้บริการ Platform ของเรา เป็นการแสดงถึงว่าคุณได้ยืนยันว่าตนเองมีอายุผ่านเกณฑ์ และมีคุณสมบัติครบที่จะทำการยอมรับตกลง และสัญญาระหว่างคุณกับตัวบริษัท Pretshy มีสิทธิ์ที่จะระงับการใช้งานบัญชีใช้งานของคุณ หากคุณมีคุณสมบัติดังกล่าวไม่ครบถ้วน

คุณสามารถติดต่อเราผ่าน Customer Support ได้ หากคุณมีคำถามเกี่ยวกับข้อตกลงการใช้งานของเรา

ผู้ว่าจ้าง คือ ผู้ว่าจ้างงานบน Pretshy ที่ตกลงยอมรับตามข้อกำหนดของ Pretshy

ผู้ใช้ คือ บุคคลที่เข้ามาสู่เว็บไซท์ของ Pretshy และใช้บริการของ Pretshy ซึ่งรวมถึง Pretty และผู้ว่าจ้างด้วย

หน้ารายละเอียดงาน คือ หน้าที่ Pretty สามารถอธิบายถึงรายละเอียดงานที่ตนเองจะส่งมอบ ซึ่งผู้ว่าจ้างสามารถอ่านและตัดสินใจว่าจ้างจาก Pretty ดังกล่าวได้

ออเดอร์ คือ สัญญาระหว่างผู้ว่าจ้างและ Pretty หลังจากผู้ว่าจ้างตัดสินใจชำระเงินจ้าง Pretty จากหน้ารายละเอียดงาน

หน้างาน คือ หน้าที่เป็นช่องทางการสื่อสารระหว่างผู้ว่าจ้างและ Pretty สำหรับงานที่ผู้ว่าจ้างจ้าง Pretty ไป

รายได้ คือ จำนวนเงินที่ Pretty ได้รับหลังจากส่งมอบงานให้แก่ผู้ว่าจ้างเรียบร้อย

- Pretty ต้องส่งมอบงานให้สมบูรณ์ตามออเดอร์ที่ได้รับมอบหมาย Pretty ไม่สามารถยกเลิกออเดอร์ได้ ยกเว้นในกรณีจำเป็นเท่านั้น ซึ่งการยกเลิกออเดอร์โดย Pretty จะส่งผลต่อสถานะและผลรีวิวของ Pretty ด้วย

- ผู้ใช้ไม่สามารถชำระเงิน หรือรับเงินออเดอร์ผ่านวิธีอื่นนอกเหนือจากการสั่งซื้อผ่าน Pretshy เท่านั้น

- ผู้ว่าจ้างมีสิทธิ์เป็นเจ้าของงานที่ Pretty ส่งมอบทั้งหมด ยกเว้นกรณี Pretty ได้ระบุรายละเอียดของสิทธิ์การเป็นเจ้าของผลงานบนหน้ารายละเอียดงาน

- Pretshy มีสิทธิ์ที่จะนำผลงานต่างๆที่ส่งมอบบน Pretshy ไปใช้สำหรับการโฆษณาต่างๆ รวมไปถึงการตลาดต่างๆ ของ Pretshy

- เราให้ความสำคัญแก่ความปลอดภัยของข้อมูลของคุณ คุณสามารถอ่านนโยบายความเป็นส่วนตัวได้ ซึ่งนโยบายความเป็นส่วนตัวนี้ถือเป็นส่วนหนึ่งของข้อตกลงการใช้งานด้วยา


Pretty

- Pretshy จะส่งมอบรายได้ให้คุณเมื่อออเดอร์อยู่ในสถานะเสร็จสิ้นแล้วเท่านั้น คุณสามารถดูนิยามของคำว่าออเดอร์อยู่ในสถานะเสร็จสิ้นได้ที่หมวด ออเดอร์

- หากออเดอร์ถูกยกเลิก จำนวนเงินสำหรับออเดอร์นี้จะถูกส่งมอบกลับคืนสู่ผู้ว่าจ้างและ Pretty ตามจำนวนที่ตกลงเอาไว้บน Pretshy

- รายได้ของ Pretty จะได้รับการโอนอัตโนมัติตามที่กำหนดไว้

งาน

- Pretty สามารถประกาศประวัติรายละเอียดงานของตนเองบน Pretshy ได้โดยไม่เสียค่าบริการใดๆ

- ผู้ว่าจ้างและ Pretty สามารถโพสงานของตนเองบน Pretshy ได้โดยไม่เสียค่าบริการใด ๆ

- งานต่างๆที่แสดงอยู่บน Pretshy ถือว่าเป็นเนื้อหาที่ถูกสร้างขึ้นโดยผู้ใช้

- บัญชีของ Pretty อาจมีสิทธิ์ถูกระงับการใช้งาน หากงานบริการของ Pretty ถูกลบทิ้งด้วยเหตุผลตามข้อข้างต้น

- งานที่ถูกลบออกไปด้วยเหตุผลตามข้อข้างต้น จะไม่สามารถนำกลับมาแก้ไข หรือนำกลับมาใช้ในระบบได้

งานหรือ ประวัติ Pretty ที่ได้รับรีวิวที่ไม่ดีอาจไม่ถูกนำมาแสดงในระบบของ Pretshy ได้

- บัญชีของ Pretty อาจมีสิทธิ์ถูกระงับการใช้งาน หากงานบริการของ Pretty ถูกลบทิ้งด้วยเหตุผลตามข้อข้างต้น

- URLs ที่ถูกระบุเอาไว้บนหน้ารายละเอียดงานที่พาผู้ใช้ไปสู่เว็บไซต์อื่นนอกเหนือจาก Pretshy อาจะถือว่าเป็นการใช้งานระบบ Pretshy อย่างไม่เหมาะสม และต้องผ่านการอนุมัติจาก Pretshy ก่อนเท่านั้น

- ประวัติงานทุกประวัติจำเป็นต้องมีรูปภาพประกอบ โดย Pretshy จะมีช่องทางให้ Pretty สามารถอัพโหลดรูปประกอบประวัติได้

- ประวัติ Pretty สามารถมีวิดีโอที่ผ่านการอนุมัติแล้วสำหรับการเป็นวีดีโอประกอบได้

- Pretty ไม่สามารถเพิ่มเติมรายละเอียดงานที่ส่งผลกระทบแก่การเปลี่ยนแปลงข้อตกลงของ Pretshy ได้


การถอนรายได้

- ผู้ใช้ต้องกรอกข้อมูลบัญชีที่ถูกต้องเพื่อที่จะทำการถอนรายได้ได้

- ผู้ใช้สามารถถอนรายได้ได้สูงสุดเป็นจำนวนเงินที่มีอยู่ในบัญชีผู้ใช้เท่านั้น ระบบจะทำการโอนเงินคงเหลือของท่านทั้งหมดโดยอัตโนมัติ ทุกวันอังคารของสัปดาห์โดยไม่มีค่าธรรมเนียมการโอนเงิน

- ผู้ใช้ไม่สามารถยกเลิกการถอนรายได้ได้ และไม่สามารถขอถอนเงินรายได้ก่อนวันที่กำหนดได้

- ไม่มีค่าใช้จ่ายใดใดในการถอนรายได้


ผู้ว่าจ้าง

- ผู้ว่าจ้างไม่สามารถชำระเงินให้ Pretty ผ่านช่องทางอื่นนอกจากช่องทางที่ Pretshy ได้กำหนดให้บน Platform Pretshy

การซื้อ

- ผู้ว่าจ้างไม่สามารถชำระเงินสำหรับงานบน Pretshy ผ่านช่องทางอื่นนอกเหนือจากช่องทางที่ Pretshy กำหนดให้เท่านั้น กรุณาแจ้งมาที่เรา หากมี Pretty ขอให้คุณชำระเงินผ่านช่องทางอื่นนอกเหนือจากช่องทางที่ Pretshy กำหนดให้

- Pretshy จะไม่มีการเก็บข้อมูลบัตรเครดิตของผู้ใช้เพื่อป้องกันการโกง หรือการถูกขโมยข้อมูล แต่ Pretshy อนุญาตให้บริษัทให้บริการการชำระเงินที่ Pretshy ใช้เก็บข้อมูลบัตรเครดิตต่างๆเพื่อใช้ในการชำระเงินบน Pretshy ซึ่งการใช้ข้อมูลดังกล่าวจะถูกผูกมัดภายใต้ข้อตกลงการใช้งาน และนโยบายความเป็นส่วนตัวของบริษัทให้บริการชำระเงิน


ออเดอร์

ข้อตกลงทั่วไป

- หลังจากระบบได้ทำการยืนยันการชำระเงินแล้ว ระบบจะทำการสร้างออเดอร์ให้คุณโดยจะมีหมายเลขออเดอร์กำกับ

- Pretty ต้องส่งมอบงานหรือบริการโดยใช้ปุ่ม ส่งงานขั้นสุดท้าย (อยู่บนหน้างาน) ให้แก่ผู้ว่าจ้างที่ชำระเงินสำหรับงานดังกล่าวตามรายละเอียดงานที่ระบุเอาไว้บนหน้ารายละเอียดงาน

- การใช้ปุ่ม ส่งงานขั้นสุดท้าย ในทางที่ไม่สมควร เช่นใช้ส่งงานที่ยังไม่สมบูรณ์ หรือส่งงานที่ไม่เป็นไปตามข้อตกลง จะส่งผลให้ Pretty ถูกฟ้องร้องงาน หรือคะแนนรีวิวของ Pretty ลดลง หรือ Pretty อาจถูกระงับบัญชี Pretshy ได้

เนื้อหาที่สร้างโดยผู้ใช้ (User Generated Content หรือ “UGC”) หมายถึงเนื้อหาต่างๆที่ถูกสร้างขึ้นมาโดยผู้ใช้อื่นๆในระบบที่ไม่ได้ถูกสร้างจากตัว Pretshy หรือ Platform Pretshy เอง
